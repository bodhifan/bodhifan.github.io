2. 说明

2.1 下载repo

克隆github项目: git clone https://github.com/bodhifan/Hexo.git

2.2 安装环境

安装nodejs环境;npm 包管理工具：详细过程见 个人博客的搭建

安装hexo cli : npm install hexo-cli -g

安装 依赖包：npm install
